## نمای کلی

**سرای هنر و اندیشه** یک اپلیکیشن وب فارسی‌زبان هست که برای اشتراک‌گذاری محتوا، پادکست، ویدیو، کتاب و گفتگوی جمعی طراحی شده. رابط کاربری کاملاً راست‌چین و بهینه برای موبایل و دسکتاپ ساخته شده.

## امکانات

### محفل (گفتگوی جمعی)
- ارسال متن، عکس، ویدیو و صوت
- سیستم ریپلای و پاسخ به پیام‌ها
- لایک و حذف پیام‌ها
- ویرایش پیام‌ها
- حالت چت سبک ایتا با حباب‌های رنگی
- صفحه گفتگوی اختصاصی برای هر پست یا کامنت

### پخش رسانه
- پخش کننده ویدیوی سفارشی با کنترل‌های لمسی و کیبورد
- پادکست با لیست پخش و اپیزودها
- پخش صوتی در پس‌زمینه
- حالت مینی پلیر

### محتوا
- یادداشت‌ها و پست‌های متنی
- نشر (کتاب‌ها و یادداشت‌ها)
- کتابخانه شخصی
- جستجوی سراسری

### احراز هویت
- ورود با OTP (پیامکی)
- پروفایل کاربری با نقش‌ها (کاربر، مدیر)

### تعاملات
- لایک پست‌ها و کامنت‌ها
- ذخیره ویدیو در کتابخانه
- اشتراک‌گذاری
- منوی تعاملی با کلیک راست

## راه‌اندازی

### پیش‌نیازها
- Node.js 18 یا بالاتر
- MongoDB

### اجرای محلی

```bash
# نصب وابستگی‌ها
npm install

# اجرای سرور بک‌اند
cd server
node server.js

# در ترمینال دیگر، اجرای فرانت‌اند
npm run dev
```

اپلیکیشن در `http://localhost:3000` در دسترس خواهد بود.
سرور API در `http://localhost:5000` اجرا می‌شود.

### ساخت برای تولید

```bash
npm run build
```

فایل‌های بهینه‌شده در پوشه `dist/` ذخیره می‌شوند.

## محیط

 متغیرهای محیطی در فایل `server/.env`:

```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/soha
JWT_SECRET=your_secret_key
JWT_EXPIRES_IN=7d
```

## ویژگی‌های فنی

- **کشینگ سمت سرور**: هدر `Cache-Control` برای درخواست‌های GET API
- **فشرده‌سازی**: Gzip و Brotli برای فایل‌های ساخت‌شده
- **Code Splitting**: جداسازی React از بقیه کدها در build نهایی
- **ایندکس‌گذاری MongoDB**: ایندکس‌های بهینه برای کوئری‌های پرتکرار
- **رابط کاربری واکنش‌گرا**: طراحی mobile-first با پشتیبانی دسکتاپ
- **حالت تاریک**: پشتیبانی کامل از تم تاریک و روشن
