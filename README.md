> ЩҫШ®ШҙвҖҢЪ©ЩҶЩҶШҜЩҮ ШөЩҲШӘЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ ШЁШ§ ЩӮШ§ШЁЩ„ЫҢШӘ ШІЩ…Ш§ЩҶвҖҢШЁЩҶШҜЫҢШҢ ЩҫШ®Шҙ ШҜШұ ЩҫШівҖҢШІЩ…ЫҢЩҶЩҮ ЩҲ Щ„ЫҢШіШӘ ЩҫШ®Шҙ ШіЩҒШ§ШұШҙЫҢ

- ЩҫШ®ШҙвҖҢЪ©ЩҶЩҶШҜЩҮ ШөЩҲШӘЫҢ Ъ©Ш§Щ…Щ„ ШЁШ§ Ъ©ЩҶШӘШұЩ„вҖҢЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ
- ЩҫШ®ШҙвҖҢЪ©ЩҶЩҶШҜЩҮ Ъ©ЩҲЪҶЪ© (Mini Player) ШҜШұ ЩҫШ§ЫҢЫҢЩҶ ШөЩҒШӯЩҮ
- ЩҫШ®Шҙ ШӘЩ…Ш§Щ…вҖҢШөЩҒШӯЩҮ ШЁШұШ§ЫҢ ШӘШ¬ШұШЁЩҮ ШЁЩҮШӘШұ
- Щ„ЫҢШіШӘ ЩҫШ®Шҙ (Playlist) ШЁШ§ Щ…ШҜЫҢШұЫҢШӘ ШўШіШ§ЩҶ
- Ш¬Ш§Щ…Ш№ЩҮ ЩҶШёШұШ§ШӘ ЩҲ ШӘШ№Ш§Щ…Щ„ Ъ©Ш§ШұШЁШұШ§ЩҶ
- ШўЩҫЩ„ЩҲШҜ ЩҲ Щ…ШҜЫҢШұЫҢШӘ ЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ ШөЩҲШӘЫҢ

- ЩҫШ®ШҙвҖҢЪ©ЩҶЩҶШҜЩҮ ЩҲЫҢШҜЫҢЩҲЫҢЫҢ ШЁШ§ Ъ©ЫҢЩҒЫҢШӘ ШЁШ§Щ„Ш§
- ЩҫШ®Шҙ ШўЩҶЩ„Ш§ЫҢЩҶ YouTube ШЁШ§ Embed
- ЩҫШ®ШҙвҖҢЪ©ЩҶЩҶШҜЩҮ Ъ©ЩҲЪҶЪ© ЩҲЫҢШҜЫҢЩҲЫҢЫҢ
- Щ„ЫҢШіШӘ ЩҲЫҢШҜЫҢЩҲЩҮШ§ ШЁШ§ ЩҒЫҢЩ„ШӘШұ ЩҮЩҲШҙЩ…ЩҶШҜ
- ЩҫШ®Шҙ ШӘЩ…Ш§Щ…вҖҢШөЩҒШӯЩҮ ЩҲ ШӘЩ…Ш§Щ…вҖҢШөЩҒШӯЩҮ ЩҲШ§ЩӮШ№ЫҢ
- Ш¬Ш§Щ…Ш№ЩҮ ЩҶШёШұШ§ШӘ ЩҲЫҢШҜЫҢЩҲ

- ШіШЁШҜ Ш®ШұЫҢШҜ ЩҲ ЩҫШұШҜШ§Ш®ШӘ ШўЩҶЩ„Ш§ЫҢЩҶ
